This is a normal paragraph. You can use many types of text styling within it. For example, this is a strong text and this an emphasized text. These kind of text stylings replaces old protocols like bold and italics.

The next thing we are going to put in this page is an ordered list, using the tag "ol". Every ordered list has list items, which must be declared by using the tag "li". For this example, we are going to build a list with the name of five colors.

  1. List of Colors
  2. Red
    • Crimson
    • Blood
    • Ruby
    • Sunset
  3. Orange
  4. Green
  5. White
  6. Black

The following exercise would be creating an unordered list. Instead of using numbers from 1 to 5 like in the list above, an unordered list will show bullet points with no specific hierarchy. List items must be declared with the "li" tag too.

  • List of Colors
  • Red
    • Crimson
    • Blood
    • Ruby
    • Sunset
  • Orange
  • Green
  • White
  • Black
